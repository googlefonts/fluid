Variable Fonts
Variable fonts are the latest evolution of fonts.

This new format allows users to create custom variations of a typeface from a single, small file instead of having a separate font file for every style, width, or weight.

Optical size & Grade

Optical size is used to give text a more natural appearance by optimizing the readability of letterforms for the font size.

Grade changes the thickness of text without changing the width.
